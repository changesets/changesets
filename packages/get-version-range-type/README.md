# @changesets/get-version-range-type

[![Open on npmx.dev](https://npmx.dev/api/registry/badge/version/@changesets/get-version-range-type?name=true)](https://npmx.dev/package/@changesets/get-version-range-type)
[![View changelog](https://npmx.dev/api/registry/badge/version/@changesets/cli?color=229fe4&value=View+changelog&label=+)](./CHANGELOG.md)

Simple function that takes in a string which is a version range (ie `^1.0.0`, `~3.5.1`, `2.0.0`)
and returns the range definition.

A helper published mostly to ensure consistency across several changeset packages.
